BLAST
This Basic local alignment search tool uses the NCBI blast application which allows you to search your one or more query against our Mango Genome Database. Please click on the red link to select the option, you'll be asked for the query sequence and other parameters in the selected page.

Please select the appropriate choice:

Query Type Database Type BLAST Program
Nucleotide Nucleotide blastn: Search a nucleotide database using a nucleotide query.
Protein Protein blastp: Search protein database using a protein query.

References

  • Altschul, S. F., Gish, W., Miller, W., Myers, E. W., & Lipman, D. J. (1990). Basic local alignment search tool. J Mol Biol, 215(3), 403-410.
