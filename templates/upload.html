Phylogenetic Tree Building

Phylogenetic Tree

A phylogenetic tree or evolutionary tree is a branching diagram or "tree" which shows the inferred evolutionary relationships among various biological species or other entities. The tree is build upon similarities and differences in their physical or genetic characteristics. This program requires multiple sequences in FASTA format to do the alignment using clustalw algorithm and will show the results which can be further modified and processed to get the desired information or picture. It aims to provide a common way to work with phylogenetic trees independently of the source data format.
